# Changelog

## 0.1

First public release.

- Shared Codex pet atlas math in `@petx/core`.
- Framework renderers for React, React Native, Vue, Svelte, SolidJS, and Web Components.
- Shared CSS renderer for 8 x 9 Codex pet spritesheets.
- Example site with playground, props reference, framework snippets, and atlas preview.
